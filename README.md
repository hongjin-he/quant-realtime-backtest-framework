## ✨ Features
- ⚡ **Blazing Fast**: 100K+ events/sec (Numba)
- ⚔️ **Multi-Strategy Battles**
- 📊 **Pro Metrics**: PnL, Sharpe, Spread Capture, Adverse Selection
- 🎬 **Plotly Replay Animations**
- 📡 **Real Data**: Binance/Tardis replay
- 🔧 **CLI Ready**: \`pip install lob-arena\`

## 🚀 Quick Start
\`\`\`bash
pip install lob-arena
python -m lob_arena.cli battle --strategies mm,momentum --steps 10000
